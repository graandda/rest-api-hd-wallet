# eth hd wallet


## Installation

Docker must be already installed

#### Step 1 - Git

```shell
git clone https://github.com/graandda/rest-api-hd-wallet.git
```

#### Step 2 - environ setup
* rename .env.sample -> .env
* install your variables into .env

#### Step 3 - Run Docker
```shell
docker-compose build
docker-compose up
```

#### Step 4 - Enjoy!
## Features

* HD wallets creation
* Send transactions
* Filter transactions
